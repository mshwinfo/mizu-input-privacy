謎之輸入法

隱私政策 Privacy Policy ・ 最後更新日期 Last updated：2026-07-17

繁體中文

總覽

謎之輸入法（以下稱「本輸入法」）尊重並保護使用者隱私。本輸入法為完全離線運作的輸入法，不會收集、不會上傳、不會分享您輸入的任何內容。主程式首次啟動時會顯示本政策摘要，需您同意後才能繼續使用。

我們不做的事

  • 不收集您輸入的文字：僅用於當下的輸入轉換，不會被記錄、傳送或儲存到任何伺服器。
  • 不連網、不上傳：核心功能不需要網路權限，不會將任何資料傳送到外部伺服器或雲端。
  • 無帳號、無登入：不要求註冊或登入，不收集任何個人身分資訊。
  • 無廣告、無第三方分析 SDK。

儲存在您裝置本機的資料

以下資料僅儲存在應用程式私有空間，不會離開您的裝置，解除安裝 App 即一併移除：

  • 輸入偏好設定（震動／按鍵音、音量與強度、簡繁、模糊音、啟用的語言、鍵盤底圖等）
  • 使用者詞庫學習（本機記錄選字頻率，提升選字準確度）
  • 剪貼簿歷史（預設關閉，僅在您主動開啟後才記錄，最近 20 筆、可清除；關閉時不會在背景讀取剪貼簿）
  • 最近使用的表情符號（最近 24 筆）
  • 您自行匯入的碼表／詞庫／聯想詞補充表檔案
  • 聯想詞學習（本機記錄您實際採用過的聯想接續，讓常用接續詞更快出現）
  • 您選擇的鍵盤底圖
  • 您於「備份與還原」主動匯出的設定與學習資料備份檔（JSON），本輸入法不會自動上傳或分享，匯入時也僅在您主動選擇檔案後才讀取

權限說明

  • 震動：按鍵回饋，可於設定關閉
  • 剪貼簿存取：僅於您主動複製／貼上時使用，內容不離開裝置
  • 相簿存取：僅於您點擊「選擇底圖」時透過系統選擇器挑選單張圖片

關於「完整體驗模式」

HarmonyOS 會對所有第三方輸入法顯示是否啟用「完整體驗模式」的系統級提示（非本輸入法特有的行為）。系統未授予此模式時，會限制部分能力（例如按鍵震動）；開啟後這些功能才能正常運作。該提示顯示的「可能傳輸您輸入的內容」為系統對此模式的通用制式警語（所有第三方輸入法皆顯示相同文字），並非本輸入法會如此運作——本輸入法完全離線、未宣告任何網路權限，技術上不可能傳送任何資料。

外部連結

「哪裡找碼表？」與「支持開發者」的連結會開啟瀏覽器前往第三方網站（GitHub、Ko-fi、綠界科技），本輸入法不經手您在該處的任何行為，該網站有其自身的隱私政策。

兒童隱私

本輸入法不針對兒童設計，也不會刻意收集任何人的個人資訊。

English

Overview

mizu input (“the IME”) respects and protects user privacy. The IME operates fully offline and does not collect, upload, or share anything you type. The main app shows a summary of this policy on first launch and requires your consent before you can continue.

What we do NOT do

  • We do not collect what you type — used only for on-device conversion, never logged or transmitted.
  • No network, no upload — core functionality needs no network permission.
  • No account, no login.
  • No ads, no third-party analytics SDKs.

Data stored locally on your device

  • Input preferences (haptics/sound, volume/intensity, Simplified output, fuzzy phonetics, enabled languages, keyboard background, etc.)
  • User dictionary learning (local selection frequencies)
  • Clipboard history (off by default; only recorded after you enable it, last 20, clearable; no background clipboard reading while disabled)
  • Recently used emoji (last 24)
  • Code tables / dictionaries / supplementary association word lists you import yourself
  • Association learning (locally recorded associations you actually pick, so frequently-used follow-up words surface faster)
  • The keyboard background image you choose
  • A backup file (JSON) you export from "Backup & Restore", containing the same local data above; the IME never uploads or shares it automatically, and only reads a file you actively pick when importing

Permissions

  • Vibration: key haptic feedback, can be disabled in settings
  • Clipboard access: only when you actively copy/paste; content never leaves the device
  • Photo access: only when you tap "Choose background", via the system photo picker for a single image

About "Full Experience Mode"

HarmonyOS shows a system-level prompt asking whether to enable "Full Experience Mode" for all third-party input methods (not specific to this IME). When not granted, some capabilities (e.g. key vibration) are restricted; enabling it allows them to work normally. The prompt's warning that the app "may transmit anything you type" is HarmonyOS's standard boilerplate for this mode (shown identically for every third-party keyboard) — it does not describe what this IME actually does. This IME is fully offline and declares no network permission, so it is technically incapable of transmitting any data.

External links

Links under "Where to find code tables" and "Support the developer" open your browser to third-party sites; the IME is not involved in anything you do there.

Children's privacy

The IME is not directed at children and does not knowingly collect anyone's personal information.
